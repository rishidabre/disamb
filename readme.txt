Readme
------
1. Update the paths of the files 'Authors.txt', 'PaperAuthorAffiliations.txt' and 'PaperReferences.txt' in the file 'algo.py'.
2. Make sure the machine has internet connectivity.
3. Run the script at the shell prompt as follows:
	$ ./algo.py
4. At the prompt that appears, enter the password of the Neo4j user and hit enter.
